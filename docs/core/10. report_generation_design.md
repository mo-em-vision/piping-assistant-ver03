# Report Generation Design

## Purpose

Convert deterministic execution results into human-readable engineering documents while preserving traceability, auditability, and engineering correctness.

> The report explains engineering decisions; it never changes engineering truth.

---

# Pipeline

```
Task state + execution/validation/graph traces + referenced knowledge
↓
ReportData builder (engine/reports/report_data.py)
↓
Deterministic report generator (templates / formatters)
↓
Optional AI Presentation Layer
↓
PDF / HTML
```

**Flow Guidance** (`GuidanceResolver`, `ResponseComposer`) is for the **live workflow interface only** — not part of this pipeline. See [`docs/rules.md`](../rules.md) §21.

---

# Component roles

| Component | Role |
| --- | --- |
| Execution Layer | Produces traces, node results, warnings — does not build `ReportData` directly |
| ReportData builder | Assembles report-ready structure from task state + traces + knowledge refs |
| Report Generator | Templates, formatters, PDF/HTML |
| AI Presentation | Optional wording polish on completed `ReportData` |

`ReportData` must not be modified by AI except presentation fields explicitly designated for prose.

---

# ReportData contents

- Task and workflow context
- Graph traversal / execution order
- Inputs (Facts), formulas, calculations, decisions
- Warnings, limitations, standard references
- Validation trace and overrides (where recorded)

Regeneration from stored `ReportData` must not require re-execution.

---

# AI presentation limits

The AI may: improve readability, organize sections, summarize reasoning.

The AI may not: change inputs, outputs, formulas, PASS/FAIL, warnings, or standard references.

---

# Trace chain

Every report preserves:

`standard paragraph → dependency → decision → formula → output → conclusion`

Use **graph traversal trace** for expansion/navigation history and **execution trace** for deterministic runs. The Graph Engine does not execute calculations.

---

# Report types

- Engineering calculation report
- Integrity / assessment report
- Audit-oriented trace appendix
- Summary report (condensed)

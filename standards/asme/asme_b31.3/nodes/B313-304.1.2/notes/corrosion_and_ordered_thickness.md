# Corrosion and Ordered Thickness

The calculated value `t` from this node is **pressure design thickness**, not the ordered pipe wall thickness.

Apply §304.1.1 eq. (2) to obtain `t_m = t + c`, then select pipe with wall thickness `T` not less than `t_m`.

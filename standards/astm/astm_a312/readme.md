# ASTM A312/A312M — Austenitic Stainless Steel Pipe

Material property reference pack for **ASTM A312** seamless and welded austenitic stainless steel pipe grades.

> Development reference data structured from ASTM A312/A312M minimum requirements. Verify against the licensed standard for production use.

## Contents

| File | Description |
|------|-------------|
| `standards_tables.db` | Chemical composition limits, minimum tensile/yield strength, elongation, physical properties |

## Covered grades

- **TP304**, **TP304L**, **TP304H**
- **TP316**, **TP316L**, **TP316H**
- **TP321**, **TP347**, **TP317L**

## Properties included

- Chemical composition limits (%)
- Minimum tensile and yield strength at room temperature (ksi and Pa)
- Minimum elongation in 2 in.
- Physical properties (density, modulus of elasticity, thermal expansion)
- Selected elevated-temperature reference yield values for design screening (TP304, TP316, A106 Gr B)

## Related standards

- **ASME B36.10M** — pipe dimensions (`standards/asme/asme_b36.10/`)

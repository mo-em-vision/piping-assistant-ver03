# ASTM A106/A106M — Seamless Carbon Steel Pipe

Material property reference pack for **ASTM A106** grades used in piping applications (often referenced as **SA-106** in ASME codes).

> Development reference data structured from ASTM A106/A106M minimum requirements. Verify against the licensed standard for production use.

## Contents

| File | Description |
|------|-------------|
| `tables/material_properties.yaml` | Chemical composition limits, minimum tensile/yield strength, elongation, physical properties |

## Covered grades

- **A106 Gr A** — 48 ksi min tensile, 30 ksi min yield
- **A106 Gr B** — 60 ksi min tensile, 35 ksi min yield
- **A106 Gr C** — 70 ksi min tensile, 40 ksi min yield

## Related standards

- **ASME B31.3** — allowable stress tables may reference SA-106 (`standards/asme/asme_b31.3/`)
- **ASME B36.10M** — pipe dimensions (`standards/asme/asme_b36.10/`)
